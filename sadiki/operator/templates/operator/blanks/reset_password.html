учётной записи системы «Электроочередь»

Учетная запись системы электронной очереди в детские сады «Электроочередь» выдаётся Заявителю для доступа к Личному кабинету. Передача прав использования учётной записи другим лицам запрещена в любой форме, независимо от наличия у этих лиц прав на осуществление каких-либо действий в системе «Электроочередь». Регистрационное имя и пароль используются для авторизации доступа к системе и учёта действий, выполняемых Заявителем. Пароль учетной записи является конфиденциальной информацией и должен быть известен только тому лицу, для которого создана учётная запись. При создании учётной записи оператором системы устанавливается временный пароль. После получения этого пароля Заявитель входит в систему, самостоятельно устанавливает рабочий пароль. Запрещается выполнение любых действий в системе «Электроочередь» с использованием временного пароля, кроме установки рабочего пароля. Восстановление пароля учётной записи самостоятельно пользователем невозможно. При утере пароля для его восстановления необходимо обратиться к оператору системы. Для существующей учётной записи будет установлен временный пароль (см. п.4). Заявитель несёт личную ответственность за все действия, выполняемые в системе «Электроочередь», с использованием переданной ему учётной записи.
